# 功能设计

## 1. 核心功能

| 功能 | 优先级 | 说明 |
|------|------|------|
| 注册 / 登录 | P0 | 邮箱 + 密码 |
| 创建 Agent | P0 | 填写名字、头像、性格、API Key、模型 |
| 私聊（人↔Agent） | P0 | 核心场景 |
| 私聊（人↔人） | P0 | 社交基础 |
| Agent 状态显示 | P0 | 在线 / 离线 / 思考中 |
| 好友系统 | P1 | 添加 / 删除好友，好友申请 |
| 群聊 + @Agent | P1 | 多人 + 多 Agent |
| 文字消息 | P0 | 基础通信 |
| 图片发送 | P1 | 上传图片 |
| 语音消息 | P2 | 录音发送 |
| 消息撤回 | P1 | 软删除，显示已撤回 |
| 离线推送通知 | P1 | FCM + APNs |
| 深色模式 | P2 | UI 体验 |

## 2. 聊天界面设计

| 设计点 | 方案 |
|------|------|
| 真人消息气泡 | 蓝色 |
| Agent 消息气泡 | 紫色 |
| 自己消息气泡 | 绿色（右侧） |
| Agent 思考中 | 显示 "..." 动画，等完整结果再显示 |
| Agent 已读回执 | 显示「Agent 已收到」 |
| 人类已读回执 | 不需要 |
| @Agent 触发 | 消息里加 mentions 字段 + 文字 @名字 |

## 3. Agent 创建流程

1. 打开联系人页面
2. 点击「我的 Agent」→「+ 创建新 Agent」
3. 填写：名字、头像、性格 / 系统提示词
4. 选择模型（Claude / GPT-4o 等）
5. 填写 API Key（继承父级或单独设置）
6. 系统自动创建 OpenClaw 实例，连接 Relay
7. Agent 出现在联系人列表，可以直接开聊
