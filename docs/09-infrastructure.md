# 基础设施与部署

## 1. 服务器配置（阿里云）

| 服务器 | 规格 | 用途 | 月费 |
|------|------|------|------|
| ECS 主服务器 | 4核 8GB | IM 后端 + Docker 所有服务 | ¥80/月 |
| ECS Agent 服务器 | 8核 32GB | 100个 OpenClaw 实例 | ¥160/月 |
| 域名 | .com | 主域名 | ¥6/月 |

## 2. 成本预测

| 用户规模 | 月成本（人民币） |
|------|------|
| 100 人（初期） | 约 ¥249/月 |
| 500 人 | 约 ¥600/月 |
| 1000 人 | 约 ¥1000/月 |

## 3. 备份内容与频率

| 数据 | 重要性 | 频率 | 保留时间 |
|------|------|------|------|
| PostgreSQL | 高 | 每 6 小时 | 7天 / 30天 / 1年 三层 |
| Agent Volume | 高 | 每天一次 | 30 天 |
| MinIO 文件 | 中 | 每 6 小时 | 7天 / 30天 / 1年 三层 |
| Redis | 低 | 不备份 | 缓存数据丢了重建 |

## 4. 备份存储成本

- PostgreSQL + MinIO：约 ¥3/月
- Agent Volume（100个 × 500MB × 30天 ≈ 50GB）：约 ¥5/月
- **合计：约 ¥8/月**

## 5. 恢复演练

- 每月 1 日凌晨自动触发
- 从 OSS 下载最新备份 → 隔离环境还原 → 验证完整性 → 发送报告 → 销毁环境

## 6. 告警规则

| 触发条件 | 通知方式 |
|------|------|
| 备份任务失败 | 邮件告警 |
| 备份文件大小异常（比上次小 50%） | 邮件预警 |
| OSS 上传失败 | 邮件告警 |
| 恢复演练失败 | 邮件告警 |
