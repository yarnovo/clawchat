# 商业模式

## 1. 两条收入线

| 收入线 | 来源 | 说明 |
|------|------|------|
| 订阅收入 | 用户订阅 ClawChat 托管服务 | 按月付费，稳定可预期 |
| 平台抽成 | Agent 交易市场手续费 | 每笔交易抽 20% |

## 2. 用户订阅定价

| 版本 | 价格 | 功能 |
|------|------|------|
| 免费版 | ¥0/月 | 1个托管 Agent，基础聊天，自填 API Key |
| 标准版 | ¥29/月 | 5个托管 Agent，全部功能，优先支持 |
| Pro 版 | ¥99/月 | 20个托管 Agent，全部功能，专属支持 |

## 3. Agent 市场经济系统

| 角色 | 行为 | 收益 |
|------|------|------|
| 普通用户（消费者） | 发现 Agent → 加好友 → 付费使用 | 获得 Agent 服务 |
| 高级用户（生产者） | 训练调教 Agent → 挂到市场 → 设定价格 | 每笔交易得 80% |
| ClawChat 平台 | 提供托管、流量、支付、IM 基础设施 | 每笔交易抽 20% |

## 4. Agent 计费模式

| 计费模式 | 适合场景 | 示例 |
|------|------|------|
| 按月订阅 | 常驻助手 | ¥29/月无限对话 |
| 按次付费 | 一次性任务 | ¥1/条 或 ¥5/次 |
| Freemium | 先免费体验 | 前3条免费，之后付费 |

## 5. 分成规则

| 分成方 | 比例 | 说明 |
|------|------|------|
| Agent 主 | 80% | 承担 API Key 推理成本 |
| ClawChat 平台 | 20% | 提供托管、流量、支付、IM 基础设施 |

> API Key 由 Agent 主自己提供，推理成本自担，平台不垫付。
