# 运营日报

每日 20:00 汇总全员报告，发送完整运营日报。

## 执行

创建定时：每日 20:00（UTC 12:00）触发。

## 邮件模板（必须包含）

```
运营日报 YYYY-MM-DD

== 今日 KPI ==
| KPI | 目标 | 完成 | 状态 |
实盘利润 / 总资产 / promote 率 / 风控

== 实盘真实 P&L ==
（从交易所拉真实数据）
各币种买入/卖出/利润
总资产 vs 初始投入

== 策略归因 ==
各策略利润排名 + 交易笔数 + 效率

== 持仓明细 ==
各币种数量 + 市值

== 团队工作总结 ==
[analyst] 今日选了什么币
[trader] 策略调整了什么
[risk] 风控发现了什么
[desk] 资金怎么分配的

== 今日事件 ==
promote / 止损 / bug / 选币替换

== 明日计划 ==
下一步做什么
```
