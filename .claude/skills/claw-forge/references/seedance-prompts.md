# Seedance 视频提示词指南

## 黄金公式

```
[主体] + [动作] + [场景] + [风格] + [情绪/镜头运动]
```

示例：一只机械蜂鸟在霓虹灯城市上空悬停，翅膀高速振动，赛博朋克风格，镜头缓慢推进

## 镜头运动词汇

| 中文 | 英文 | 效果 |
|------|------|------|
| 推进 | dolly in / push in | 镜头靠近主体，增强紧迫感 |
| 拉远 | dolly out / pull out | 镜头远离主体，展示全貌 |
| 旋转 | orbit / rotate | 围绕主体旋转，增加动感 |
| 俯拍 | top-down / bird's eye | 从上方俯视，全景展示 |
| 仰拍 | low angle | 从下方仰视，增强气势 |
| 跟随 | tracking shot | 跟随主体运动 |
| 一镜到底 | one-shot / long take | 不剪辑的连续镜头 |
| 横移 | pan / truck | 水平移动镜头 |
| 升降 | crane / boom | 垂直移动镜头 |

## 风格关键词

- **写实 (photorealistic)** — 真实质感，适合产品展示
- **赛博朋克 (cyberpunk)** — 霓虹灯、暗色调、科技感
- **简约科技 (minimal tech)** — 干净、白色背景、线条感
- **扁平动画 (flat animation)** — 2D 风格、色块、简洁
- **未来感 (futuristic)** — 全息投影、悬浮元素
- **玻璃形态 (glassmorphism)** — 半透明、模糊背景、折射

## 图生视频最佳实践

1. **首帧提示词末尾加"视频静帧画面"** — 让生成的图片更适合作为视频起始帧
2. **首帧构图预留动态空间** — 不要把主体放在画面边缘
3. **视频提示词与首帧保持一致** — 描述同一个场景的动态变化
4. **动作要明确具体** — "缓慢旋转" 比 "运动" 好
5. **避免过于复杂的场景** — 5 秒内展示 1-2 个核心动作

## 注意事项

- 不要在提示词中要求文字出现（Seedance 不擅长生成文字）
- 避免同时描述太多运动（容易生成混乱画面）
- 人物面部生成质量不稳定，优先使用物体/场景
- 16:9 横屏适合展示，9:16 竖屏适合短视频
- 5 秒视频建议只包含 1 个核心动作 + 1 个镜头运动

## 宣传视频套路

### 科技产品发布

```
一个发光的[产品]悬浮在深色背景中，缓慢旋转，
表面有流动的光线，简约科技风格，镜头缓慢推进
```

### 功能展示

```
[主体]在操作台上，周围环绕着全息数据面板，
手指触碰面板触发波纹效果，未来感风格，一镜到底
```

### 品牌宣传

```
[品牌元素]从粒子中汇聚成形，背景是星空/城市天际线，
粒子流动汇聚，赛博朋克风格，镜头从远到近推进
```
