# 启动运营团队

`/team` 创建一个完整的 4 人运营团队，各司其职，协同工作。

## 团队成员

| 角色 | 名称 | 职责 |
|------|------|------|
| 策略研发 | **strategist** | 研究信号模型、回测验证、优化参数、产出策略文件 |
| 交易员 | **trader** | 执行交易计划、管理仓位、盯盘、资金调度 |
| 风控员 | **risk** | 止损执行、仓位监控、爆仓保护、复盘审计 |
| 技术开发 | **engineer** | 系统搭建、引擎编译、监控部署、故障排查 |

## 执行

### Step 1: 创建团队

```
TeamCreate(team_name="clawchat-ops")
```

用 Agent 工具创建 4 个成员，每人给出明确的角色定义和初始任务。

### Step 2: 初始化各角色

**strategist（策略研发）**
- 读取 `strategies/approved/` 已有策略
- 运行 `make scan` 发现高波动币种
- 开始回测搜索新策略

**trader（交易员）**
- 运行 `make account` 查余额
- 运行 `make pnl` 查当前盈亏
- 检查是否有运行中的持仓

**risk（风控员）**
- 运行 `make check` 检查风控状态
- 确认止损规则已就位
- 检查仓位风险

**engineer（技术开发）**
- 检查引擎编译状态 `make build`
- 检查系统连接状态
- 确认监控正常

### Step 3: 团队协作

团队启动后进入协作模式：
1. 策略研发产出策略 → 风控审核 → 交易员执行
2. 技术开发保障系统稳定运行
3. 风控实时监控，触发止损时通知全团队

## 沟通机制

- 使用 `SendMessage` 在团队成员间传递信息
- 重要事件（开仓/平仓/止损）实时通知 CEO
- 团队报告通过 `make notify` 发送邮件
