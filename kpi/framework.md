# ClawChat 基金 KPI 体系

## 1. 收益指标

| KPI | 计算方式 | 当日目标 |
|-----|---------|---------|
| 总利润 | 所有策略利润之和 | $10 |
| 实盘利润 | 仅 live 模式策略利润 | > $0 |
| 收益率 | 总利润 / 总投入资金 | > 0.5% |
| 单策略最高利润 | max(各策略利润) | - |
| 利润集中度 | 最高策略利润 / 总利润 | < 50%（分散风险） |

## 2. 交易指标

| KPI | 计算方式 | 健康范围 |
|-----|---------|---------|
| 总交易笔数 | 所有策略交易之和 | > 50/天 |
| 有效交易率 | 有利润的交易 / 总交易 | > 30% |
| 平均每笔利润 | 总利润 / 总交易笔数 | > $0.01 |
| 交易频率 | 总交易 / 运行小时数 | > 10/小时 |

## 3. 策略组合指标

| KPI | 计算方式 | 目标 |
|-----|---------|------|
| 策略总数 | 所有运行中的策略 | 8-15 |
| 实盘策略数 | mode=live 的数量 | 逐步增加 |
| promote 率 | 已 promote / 总数 | > 30% |
| 无效策略数 | 0 交易的策略 | < 2 |
| 策略类型分布 | 网格 / RSI / 其他 | 至少 2 种 |

## 4. 资金指标

| KPI | 计算方式 | 目标 |
|-----|---------|------|
| 总投入资金 | 主账户 + 子账户 USDT | $200 |
| 实盘资金 | live 策略的 amount * grids 之和 | 逐步增加 |
| 资金利用率 | 实盘资金 / 总资金 | 30%-80% |
| 最大单策略敞口 | max(amount * grids) | < 总资金 50% |
| 子账户数 | 活跃项目数 | 2-5 |

## 5. 风控指标

| KPI | 计算方式 | 红线 |
|-----|---------|------|
| 最大单策略亏损 | min(各策略利润) | > -$5 |
| 总浮亏 | 所有未平仓的浮动亏损 | > -$20（总资金 10%） |
| 区间偏离度 | 价格距网格边界的距离 | < 5%（预警） |
| 连续亏损笔数 | 最长连续亏损交易 | < 5 |

## 6. 系统指标

| KPI | 计算方式 | 目标 |
|-----|---------|------|
| 策略进程存活率 | 存活进程 / 应启动进程 | 100% |
| 报告发送成功率 | 成功发送 / 应发送 | 100% |
| 团队成员响应 | teammates 平均响应时间 | < 2 分钟 |
