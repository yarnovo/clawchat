# AI 影视工作室 — 飞轮计划

## 飞轮

```
AI 生成短剧 → 发布到平台获取流量 → 用户订阅 → 收入买 token → 生成更多内容
     ↑                                                              |
     └────────────────────── 正向循环 ──────────────────────────────┘
```

## Phase 1: 验证内容质量（1 周）

### 目标：出一部 demo 短剧，验证 AI 视频能不能让人看下去

- [ ] 选定题材和风格（科幻/奇幻/悬疑/都市）
- [ ] 用 claw-forge pipeline 生成第一集（30-60 秒）
- [ ] 发到朋友圈/小群收集反馈
- [ ] 根据反馈迭代 prompt 和风格

### 已完成
- [x] 火山引擎 API Key 配置
- [x] Seed-2.0-pro / Seedream / Seedance 模型开通
- [x] claw-forge pipeline 冒烟测试通过（weather 宣传片）

## Phase 2: 建立 IP（1-3 个月）

### 目标：在抖音/小红书建立 AI 短剧 IP，积累首批粉丝

- [ ] 注册账号（抖音/小红书/TikTok）
- [ ] 确定 IP 名称和风格调性
- [ ] 每周发布 3-5 集短剧
- [ ] 追踪数据（播放量/完播率/关注数）
- [ ] 迭代内容策略（什么题材播放高）

## Phase 3: 订阅变现（3-6 个月）

### 目标：搭建订阅平台，月收入覆盖 token 成本

- [ ] 搭建订阅平台（小程序/App/网站）
- [ ] 定价 ¥9.9-29.9/月
- [ ] 从免费平台导流到付费平台（短视频引流 → 完整剧集付费看）
- [ ] 目标：100 订阅用户 × ¥19.9 = ¥1,990/月（覆盖 token 成本）

## Phase 4: 飞轮加速（6 个月+）

### 目标：内容自动化生产，收入持续增长

- [ ] `/loop` 自动生成内容
- [ ] 用户反馈驱动内容策略
- [ ] 多 IP 矩阵（不同题材/风格）
- [ ] 等 Seedance 2.0 API 开放后升级到长剧

## 成本估算

| 项目 | 月成本 |
|------|--------|
| 视频生成（Seedance）| ~¥300（100 条 × ¥3） |
| 图片生成（Seedream）| ~¥50（100 张 × ¥0.5） |
| 文本生成（Seed-2.0）| ~¥5 |
| **总计** | **~¥355/月** |

盈亏平衡点：**18 个订阅用户**（¥19.9/月）
